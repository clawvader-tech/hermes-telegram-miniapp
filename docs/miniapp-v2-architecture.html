Telegram Mini App v2.0.3 Architecture

FastAPI + React SPA — 10-page mobile dashboard with dual Ed25519/HMAC Telegram auth

Authentication

  • • Dual Ed25519 (primary) + HMAC-SHA256 (fallback)
  • • Bearer session tokens via /api/auth/session-token
  • • CORS restricted to known origins
  • • Per-IP rate limiting (10 fails/60s → 15min lockout)
  • • CSP headers, XSS sanitization, SRI

10-Page Dashboard

  • • Chat — SSE streaming, file attachments, abort
  • • Agents — tmux spawn/kill/message (max 5)
  • • Cron — full CRUD with pause/trigger
  • • Status, Sessions, Analytics, Logs
  • • Skills, Config, Keys — full management

Deploy Pipeline

  • • Standalone repo → deploy.sh → hermes-agent
  • • Post-merge git hook auto-redeploys
  • • Builds from source, syncs 25+ web/src/ files
  • • Pre-flight check: verifies TG initData in built JS
  • • assume-unchanged on web_server.py
